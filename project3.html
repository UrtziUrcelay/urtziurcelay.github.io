Course Description

CS 562

This courses main objective is to learn data structures, algorithms, and techniques concerned with rendering images more accurately and efficiently in interactive computer simulations and video game software.Topics include deferred shading and related multipass rendering techniques, physically based lighting, image based lighting,color spaces, global lighting and ambient occlusion, shadow map enhancements, surface reflection models (BRDFs), lighting effects, high-dynamic range, and other topics related to real-time vertex/pixel shader programming.

External libreries used:

  • OpenGL API
  • ImGui (GUI)
  • Assimp (model loading only)

Most of the models used in the demos are copyrighted and are only used for demostration purposes.

Deferred Shading pipeline

In this assigment implement a complete new framework able to render in deferred.This rendering pipeline is able to mix between deferred renderables and forward renderables, as well as multiple render passes to apply postprocess effects.
