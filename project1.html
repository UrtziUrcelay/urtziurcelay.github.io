Course Description

CS 300

This courses main objective is to learn the use of algorithms that are essential to creating photorealistic images in interactive simulations. Rendering techniques covered include texturing, illumination models, transparency, shading algorithms, mapping techniques (bump mapping, environment/reflection mapping, etc.), and shadows.

External libreries used:

  • OpenGL API
  • ImGui (GUI)
  • Assimp (model loading only)

Most of the models used in the demos are copyrighted and are only used for demostration purposes.

Basic Lighting

This is one of the assigments for the CS300 subject. This first assigment showcases the framework that was fully developed from scratch in order to support the assigments.

It also showcases the support for multiple lights and different light types, spot-light, point light and directional light. The models have selfshadowing, but no there is no support for any other type of shadows.

The controls are provided by an GUI created using ImGui, this way the edition of the light parameters and positions. Camera controls are FPS like, WASD for movement and free rotation with the right mouse click, zoom with mouse wheel scrolling.
